Repeatevolver

A tool for repeat evolution modeling

View on GitHub Download .zip Download .tar.gz

Входные данные:

  • рандомный набор букв длиной 200-300

  • симулятор эволюции рассчитанный на миллионы последовательностей

  • набор последовательностей, каждая со временем ее создания

  • Гайк: функция для определения точки в пространсте, положение в пространстве, адекватная система координат

  • хранить только дифф от изначальной
